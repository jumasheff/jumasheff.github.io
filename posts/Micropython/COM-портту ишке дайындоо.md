# COM-портту ишке дайындоо

NodeMCU жана esp8266 системасына негизделген башка микроконтроллерлерге Micropython жаздыруу COM-порт аркылуу болот.

COM-порт абалкыда root укуктарын талап кылат. Башкача айканда, ар дайым буйруктарды `sudo` деп беришибиз керек:
```bash
sudo picocom /dev/ttyUSB0 115200
```
же 
```bash
sudo esptool.py -p /dev/ttyUSB0 -b 115200 read_mac
```

## Порттун ээси

NodeMCU/esp8266 көбүнчө `/dev/ttyUSB0` аркылуу жеткиликтүү

```bash
ls -la /dev/ttyUSB0
```
деп, бул порт талап кылган колдонуучунун тобун жана колдонуучунун атын көрө алабыз. Натыйжасы мындай сыяктуу чыгат:
```bash
crw-rw---- 1 root dialout 188, 0 Dec 31 12:13 /dev/ttyUSB0
```
Мындан биз порттун "ээсинин" тобу `dialout`, аты `root` экенин билдик.

## Порттун ээлеринин бири болобуз

Колдонуучуну кандайдыр бир топко кошуу мындайча болот: `usermod -a -G <group> <user>`.
Азыр биз өзүбүздү `dialout` тобуна кошобуз:
```bash
sudo usermod -a -G dialout murat
```
Мында мен өзүмдү (`murat`) `dialout` тобуна кошуп койдум.

Эми порт менен `sudo`су жок эле иштей берсек болот:
```bash
esptool.py -p /dev/ttyUSB0 -b 115200 read_mac
```
