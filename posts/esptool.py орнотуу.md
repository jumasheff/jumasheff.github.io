# esptool.py орнотуу

*esptool.py* Python тилинде жазылган, андыктан бул тил компьютерде орнотулган болуш керек.

## Сизде Python орнотулганбы?

Орнотулгандыгын текшериш үчүн *терминалдан* кийинкини териңиз:

~~~bash
python --version
~~~

Эгерде сизде
~~~bash
Python 2.7.12
~~~
же 
~~~bash
Python 3.5.1
~~~
сыяктуу версиясын көрсөткөн жазуулар чыкса, анда компьютериңизде Python орнотулган.

Жок болсо, анда муну теребиз:
~~~bash
sudo apt-get install python
~~~

## pip орнотулганбы?

*pip* атуу программа Python тилинде жазылган программаларды компьютерибизге оңой орнотконго жардам берет.
~~~bash
pip --version
~~~

Деп терминалдан буйрук берип, *pip'тин* версиясын билип алабыз:
~~~bash
pip 9.0.1 from /home/murat/.local/lib/python2.7/site-packages (python 2.7)
~~~

Орнотулбаган болсо, анда кийинки сыяктуу жазуу чыгат:
~~~bash
The program 'pip' is currently not installed. To run 'pip' please ask your administrator to install the package 'python-pip'
~~~

Орнотуш үчүн кийинкини теребиз:
~~~bash
sudo apt-get install python-pip
~~~

## Эми esptool.py'ды орнотконго даяр болдук
~~~bash
pip install бир_нерсе
~~~
сыяктуу кылып, *pip* менен программаны орноткондо, ал глобалдык пакет катары орнотулат. Мунун бизге кереги жок, ошондуктан биз аны колдонуучунун чөйрөсүнө орнотобуз (**-\-user** деген параметрге көңүл буруңуз):
~~~bash
pip install esptool --user
~~~

*esptool.py* азыр бул жерге орнотулду:
~~~bash
~/.local/lib/python2.7/site-packages
~~~

жана аны колдонуш үчүн ар дайым могундай териш керек:
~~~bash
python ~/.local/lib/python2.7/site-packages/esptool.py ар кандай буйруктар
~~~

Бир аз ыңгайсыз болуп калды. Жумушту бир аз жеңилдетиш үчүн мындай кылалы:
1. `~/bin` папкасын түзөбүз 
2. bash-скрипт жазабыз да, анын ичине `esptool` жайгашкан жолун "катып" коёбуз, `~/bin` папкасына сактайбыз
3. Аны аткарылуучу файл кылабыз
4. `~/bin`ди `PATH`ка кошуп коёбуз
Ошондо, биз кайсы жерде турсак дагы, `esptool`ду оңой колдоно алабыз.

## esptool.py'ды ыңгайлаштыруу
### 1. `~/bin` папкасы
Терминалды ачып, кийинкини тергенде, `bin` аттуу папка `/home/murat/bin` сыяктуу жол аркылуу жеткиликтүү болуп калат:
~~~bash
mkdir ~/bin
~~~
### 2. bash script
Текст редакторун ачып, ичине төмөнкүнү жазабыз:
~~~bash
#!/bin/bash

# орнотулган жери
esptool=~/.local/lib/python2.7/site-packages/esptool.py

python $esptool
~~~
Бул файлды эми `esptool` деп атап, жаңы түзүлгөн `/home/колдонуучунун_аты/bin` папкасына сактайбыз.

### 3. жаңы түзүлгөн esptool файлын аткарылуучу кылабыз
Аткарылуучу файл кылган соң, сиз файлдын алдына `bash` же `./` деп кошумча нерсе жазып убара болбостон, `бир_скрипт` сыяктуу эле өзүнүн атын жазып иштейсиз.
`~/bin` папкасынын ичинен
~~~bash
sudo chmod +x esptool
~~~
деп тергенден кийин, `esptool` скриптибиз аткарылуучу болуп калды.

###4. `~/bin` папкасын `PATH`ка кошуу
`gedit ~/.bashrc` же `xed ~/.bashrc` (менде xed деген редактор орнотулган, мисалы) деп `.bashrc`деген жашырылган файлды ачабыз. (Мындай файл сизде жок болсо, анда ичи бош болот. Кийинкилерди аткара бериңиз). Файлдын эң төмөн жагына түшүп, кийинки сапты кошобуз жана файлды сактайбыз.
~~~bash
export PATH=$PATH:/home/колдонуучунун_аты/bin
~~~
Албетте, `колдонуучунун_аты` деген жерге компьютериңиздеги логиниңизди жазышыңыз керек.

Азыр эми `~/bin` папкасындаг бардык аткарылуучу файлдар каалаган жерден (терминал аркылуу) чакырылат.
